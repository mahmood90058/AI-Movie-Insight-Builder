# AI Movie Insight Builder

## Setup Instructions
. 1.Clone the repository:
   ```bash
   git clone https://github.com/mahmood90058/AI-Movie-Insight-Builder.git

   2.Navigate to the project folder:
   cd my repo

   3.Install dependencies:
   npm install

   4.Run the development server:
   npm run dev

   5.Open in browser: http://localhost:3000


## Tech Stack Rationale
Next.js: For React-based server-side rendering and optimized performance.

Node.js: Built-in backend via Next.js API routes.

Tailwind: CSS: For responsive and modern UI.

Grokcloud API: To summarize audience reviews and classify sentiment.


## Assumptions

Users will provide valid IMDb IDs starting with "tt".

Internet connection is required for API calls.

Secret keys stored in .env.local (not pushed to GitHub).
